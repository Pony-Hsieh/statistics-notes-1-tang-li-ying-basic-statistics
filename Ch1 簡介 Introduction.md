## 統計基本概念與專有名詞介紹
### 統計學基本概念
- 統計學主要目的(The objective of Statistics)：  
	**由樣本所得之資訊，來推論母體參數**  
  => 由樣本推論群體的表徵值
- 蒐集、整理、展示、分析、解釋 資料，並**由樣本推論群體**，使能夠在不確定的情況下完成決策的科學方法
  1. 蒐集
     - 因為要獲得群體的完整資料在實務上趨近不可能，也不實際，所以就退而求其次，抽樣(抽取樣本)得到部分的樣本數據，並透過這些樣本來推論母群體的樣貌
     - 做全檢(全數檢驗，100% inspection)不符合成本效益，所以會選擇做抽檢(抽樣檢查，抽取樣本出來檢查)，統計學建議使用抽取隨機樣本獲取資料後，再進行資料分析
  2. 整理
     - 篩選、排序、移除離群值(outlier)之後再分析
       - 統計學有一套界定離群值的方法
  3. 展示
     - 資料視覺化
       - 透過圖形，對資料有簡單的認識
  4. 分析
     - 想研究的問題，會影響要採取何種方法進行分析
  5. 解釋
     - **透過統計方法，獲得做決策的量化依據**
- 統計學的範圍
  - 敘述統計(Descriptive Statistics)
    - 蒐集數據、展示數據、找出可描述特徵值的方法
    - 歸納法
  - 推論統計(Inferential Statistics)
    - 由樣本推論群體，並估計該推論之可信度的方法
    - 演繹法
### 專有名詞
1. 群體、母體(population)
   - 由具有共同特性之個體，所組成的整體
     - 研究人員有興趣的共同特性；例如，我想知道樣本的收入與花費之間的關係，那樣本的身高、體重就不重要
2. 樣本(sample)
   - 群體的一部分
   - 隨機樣本
     - 代表性要足夠(足以由樣本代表群體的特性)
3. 參數(parameter)
   - 由 群體 資料所計算的 群體 表徵值(代表群體特徵的數值)
     - 由群體資料算出來的簡單指標、綜合指標，都可以算是參數
       - ex:
         - 群體平均數(population mean)： μ(唸 mu)
           - 樣本平均數： x̄(x-bar)
         - 群體標準差(population standard deviation)： σ(唸 sigma)
           - 樣本標準差： s
         - 群體比例： p(proportion)
           - 樣本比例： p̂(p-hat)
4. 統計量(statistics)
   - 由 樣本 資料所計算的 樣本 表徵值(代表樣本特徵的數值)
5. 隨機變數(Random Variable, R.V.)
   - 研究者對於想要研究的問題，所定義的群體，所感興趣的一項或多項特徵
     - 例如： 想要研究陽交大學生的每日手機使用時長，那 每日手機使用時長 就是 R.V.
   - 觀察各項隨機變數的結果，稱為資料
   - 不同類別的變數，就會產生不同類別的資料
   - R.V. 類別
     1. 定性變數(Qualitative)
        - 定性變數產生**類別**資料，意即 R.V. 的結果不能以數量表示，僅能依其特性的類別表示
        - 可能值： 0 或正整數
          - ex:
            - 性別(gender)
              - 即使男生設定為 1，女生設定為 0，這也不代表它算是數值資料，它只是用數學來編碼，因為在這裡的 0、1 不具備數學含意
            - 國籍(Nationality)
            - 造成產品變異的可能原因(Causes for product variation)
     2. 定量變數(Quantitative)
        - 定量變數產生**數值**資料，意即 R.V. 的各結果可以用數量表示
        - 可能值： 任何實數
        - 通常以 X 或 Y 表示
        1. 離散型資料(Discrete)： 經由 計數 的方式取得
           - 不良品個數(Defective items)
           - 一份文件之錯誤字數
           - 晶圓上的缺陷點數(Defect counts)
        2. 連續型資料(Continuous)： 經由 量測 的方式取得
           - 重量
           - 高度
           - 溫度
6. 實驗單位(Experimental Unit)
   - 研究者由所要研究的人事物上，取得隨機變數的量測值，這些人事物，稱為實驗單位
     - 我從怎樣的人事物上面，取得我要的 data，那個人事物，就是實驗單位
- 名詞比較：
  - 參數與統計量的差異：
    - 由群體算出來的指標(實務上很難取得，或者是取得成本很高) => 參數
    - 由樣本算出來的指標 => 統計量
### 進行統計分析的五大步驟
1. 定義想要研究的問題，以及該問題相關的群體
2. 規劃實驗或問卷
3. 蒐集、分析資料
4. 進行統計推論
5. 衡量統計推論的可靠度

## 2. 常用統計圖表

#### Pareto
- 與長條圖相比，主要的差異是，一定要由大到小排序，主要是為了找出影響力較大的變因，所以才要排序
- 通常會抓造成 80% 影響的變因
  - 例如
    - 因素 A: 50%
    - 因素 B: 30%
    - 因素 C: 10%
    - 因素 D: 5%
    - 因素 E: 5%
    - 那我們就只看因素 A、B
- Pareto 法則，其實就是著名的 80/20 法則

## 3. 數據取得方式
### A. 普查
- 在想要研究的群體當中，蒐集每一個體的資料，也就是 100% 全檢
### B. 抽查 (Sampling)
- 利用一種程序或方法，從群體中抽出樣本
- 重點應該就是，要想辦法讓抽出來的樣本具有代表性
- 常見的抽樣方法
  1. 簡單隨機抽樣
     - 群體中，每一個體被抽中的機會相同
     - 作法： 對群體內每一個體編號，再以亂數表、電腦模擬亂數，或製作紙籤的方式決定要抽取的樣本
     - 優點： 簡便
     - 缺點： 可能抽到樣本過於集中，導致代表性不足

     - 只在第一次做隨機抽樣，後續就依固定間隔數

  3. 分層隨機抽樣
     - 先分層，再簡單隨機抽樣
     - 先將群體依照某個衡量標準，分成數個不重疊的子群(層)，再從每一個子群(層)中，利用簡單隨機抽樣方法抽取樣本
     - 同層內的性質差異要小(類似)，不同層之間的差異要大
     - ![](<images/Ch1 簡介 Introduction/2024-06-25 (Tue) 11-15-49.png>)
  4. 部落抽樣
     - 常用在群體中的個體分離相當遠，而且很難蒐集到其樣本資料時
     - 部落抽樣假設每個部落都是群體的縮影，因此不同部落間的個體性質差異要小、同一部落內個體性質差異要大
     - 作法： 先將群體分成數個部落，再從中抽出一個或數個部落進行普查
